# 🩺 MediScope AI

Intelligent Medical Report Understanding System using NLP and Self-Attention.

## Features

* Medical Specialty Prediction
* Explainable AI
* Attention Visualization
* Positional Encoding Heatmap
* Clinical Term Analysis
* Interactive Streamlit Dashboard

## Tech Stack

* Python
* TensorFlow
* NLP
* Streamlit
* Plotly

## Model Architecture

Embedding
↓
MultiHeadAttention
↓
Dense Layers
↓
Medical Specialty Prediction

* Neurology
* Orthopedics
* Dermatology
* Radiology
